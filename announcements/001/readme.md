# Miaowing Tab 1.0.0 正式发布

Miaowing Tab 进入 1.0.0 正式版本。这个版本把桌面、设置、小组件、壁纸和数据管理重新整理到更稳定的状态，让新标签页更适合长期使用。

## 本次重点

- 主题体系完成重构，弹窗、按钮、表单和暗色模式的视觉表现更统一。
- 推荐壁纸中心支持远程更新，后续新增壁纸不需要重新发布插件。
- 小组件体验继续完善，天气、待办、日历、番茄钟、热榜等内容可以更稳定地放在桌面上。
- 自定义图标添加流程增强，支持在线图标、本地图标裁剪和纯色图标。
- 数据备份、恢复和浏览器收藏夹同步流程更清晰，重要操作前会给出更明确的确认。

## 使用建议

升级到 1.0.0 后，可以先在设置中检查主题、壁纸和桌面密度，再根据自己的使用习惯整理分类和小组件。

感谢你愿意把 Miaowing Tab 放在每天都会打开的新标签页里。后续公告会继续通过插件内通知展示，重要更新不再只依赖重新发布插件。
