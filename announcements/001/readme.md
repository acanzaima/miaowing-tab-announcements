# Miaowing Tab 1.0.0 正式发布

Miaowing Tab 1.0.0 希望把新标签页变成一个真正可以长期使用的个人桌面。它不只负责打开网站，也可以承载搜索、分类、文件夹、小组件、壁纸、诗词和数据管理这些每天都会用到的内容。你可以把它整理得很安静，只保留搜索和常用入口；也可以让它更像自己的工作台，把天气、待办、日历、热榜和常用站点都放在触手可及的位置。

![Miaowing Tab 桌面总览](./images/desktop-overview.png)

## 每天打开的第一屏

桌面是 Miaowing Tab 的中心。常用网站可以直接放在桌面上，不同场景可以放进不同分类，侧边栏用于快速切换工作、娱乐、学习或生活。多个图标也可以拖拽合并成文件夹，让同一类入口集中在一起；文件夹有不同形态，既可以节省空间，也能保留足够清楚的预览。

如果你已经有一套熟悉的排列方式，可以按分类锁定布局，避免整理好的桌面被误拖动。显示密度也可以在紧凑、标准和宽松之间切换：紧凑适合想放更多内容的屏幕，宽松适合更舒展的桌面，标准则保持日常使用的平衡。图标和小组件还可以选择顺序排列或紧凑填充，让桌面在不同内容数量下都更稳定。

添加和编辑图标也更完整。你可以从内置网址导航里挑选常用站点，也可以自己添加网站；输入网址后可以获取在线图标和站点信息，本地图标可以上传后裁剪显示区域，纯色图标则适合快速制作一个简洁的文字入口。编辑已有图标时，桌面会尽量定位到当前图标所在位置，不用再从第一屏一路翻找。

## 搜索和命令更适合高频操作

搜索框是打开新标签页后最自然的起点。你可以选择搜索引擎，开启搜索建议和搜索历史，设置结果在当前标签页或新标签页打开，也可以用 Tab 键快速切换搜索引擎。常用搜索会随着使用逐渐积累，下一次输入相同内容时更快找到目标。

命令面板则负责处理更快的插件内操作。通过快捷键打开后，可以搜索命令、切换分类、打开桌面图标、添加图标或小组件、进入编辑主页、锁定布局、切换主题、调整显示密度、进入纯净模式，也可以查看版本变化或同步浏览器收藏夹。对键盘用户来说，很多操作不需要再移动鼠标去找入口。

桌面右键菜单也覆盖了常用操作：添加内容、打开命令面板、下载当前壁纸、编辑主页、切换小组件尺寸、删除或复制图标链接，都可以在当前上下文里完成。

## 小组件成为桌面里的小应用

小组件不再只是摆在桌面上的卡片，而是可以长期停留在桌面里的小应用。天气、倒数日、待办、便签、日历、番茄钟、世界时钟、常用站点和热榜都可以添加到当前分类，并根据小、中、大尺寸展示不同层级的信息。

![添加小组件](./images/widgets-gallery.png)

天气适合快速看当前城市、温度、体感、湿度、风速和未来趋势；倒数日适合记住考试、旅行、纪念日或项目节点；待办和便签可以把临时想法、今天要做的事直接留在桌面；日历能查看月份和近期日程；番茄钟用于专注与休息；世界时钟适合关注多个时区；常用站点会根据桌面点击统计整理入口；热榜可以查看微博、知乎、百度、抖音、夸克和头条等来源的热门内容。

小组件的操作也更接近桌面直觉：点击展示区域可以打开详情，长按展示区域可以拖拽排序，只有按钮、输入框、切换项这类真正需要操作的内容才会接管点击。这样天气、倒数日、便签这类偏展示的小组件可以像桌面物件一样移动，待办、番茄钟、世界时钟这类有控件的小组件也能保留自己的交互。

## 壁纸、主题和纯净模式

壁纸中心支持系统背景、推荐壁纸、在线 URL 和本地上传。推荐壁纸可以按分类浏览，也可以随机应用；在线链接会先展示预览；本地上传会保存到浏览器中，方便下次继续使用。桌面还可以直接下载当前壁纸，看到喜欢的背景时不用再到处找来源。

![壁纸设置](./images/wallpaper-center.png)

1.0.0 重新梳理了主题视觉。浅色模式更清爽，深色模式更适合夜间使用；按钮、弹窗、表单、小组件和壁纸上的前景内容也尽量保持统一和清晰。你仍然可以按照自己的习惯控制侧边栏、设置按钮、主题按钮和纯净模式按钮的显示方式，让桌面保留需要的入口，也少一些不必要的干扰。

纯净模式适合只想保留搜索、时间和背景的场景。开启每日诗词后，页面可以展示每日推荐，也可以使用你自己整理的诗词或短句。自定义内容支持搜索、导入、随机展示和指定展示，适合放一句提醒、一段喜欢的话，或者给每天打开浏览器的瞬间留一点停顿。

## 添加内容更自由，数据也更安心

添加入口分为网址导航、小组件和自定义添加。网址导航适合快速选择内置站点，小组件适合把信息卡片安装到桌面，自定义添加则适合整理自己的工具、文档、站点或临时链接。添加成功后的提示也可以在设置中开关，如果你更喜欢安静的操作流程，可以把它关闭。

![自定义添加](./images/custom-icon.png)

数据管理在 1.0.0 中继续以“不要打扰用户已有内容”为原则。你可以手动备份当前数据；恢复备份前，可以先预览将要恢复的分类、图标、文件夹、小组件、缓存文件和搜索历史，再决定覆盖或合并。恢复前也会尽量自动保存当前状态，方便撤销上次恢复。

浏览器收藏夹同步同样会先预览。它只会新增还没有同步过的分类和收藏，不会覆盖你已经移动、删除或手动修改过的桌面内容。对已经有大量收藏夹的用户来说，这是一种更稳妥的迁移方式：先看清楚会新增什么，再决定是否同步。

![数据管理](./images/data-management.png)

## 升级建议

升级到 1.0.0 后，建议先打开设置检查主题、壁纸、显示密度、搜索和时间日期显示方式，再回到桌面整理分类、文件夹和小组件。如果你的桌面已经积累了很多内容，可以先备份一次数据，再尝试新的布局、密度和壁纸。

如果你喜欢简洁的新标签页，可以从纯净模式和每日诗词开始；如果你希望桌面承载更多信息，可以先添加天气、待办、日历、常用站点和热榜，再根据自己的使用频率调整尺寸和位置。Miaowing Tab 不需要一次整理到完美，它更适合随着每天的使用慢慢长成你的样子。

感谢你愿意把 Miaowing Tab 放在每天都会打开的新标签页里。
